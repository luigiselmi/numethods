Numerical Methods with Python
=============================
Numerical methods using Python. Examples from the book by Johansson "Numerical Python: Scientific Computing and Data Science Applications with NumPy, SciPy and Matplotlib, 3rd Edition". The book is an introduction to the Python packages used in numerical methods. The theory behind the methods implemented is explained at a very high level. Some textbooks about numerical methods that can be used with the examples are in the references.

## Notebooks
* [Ch.3 - Symbolic Computing](symbolic_computation.ipynb)
* [Ch.4 - Plotting and Visualization](plotting_and_visualization.ipynb)
* [Ch.5 - Equation Solving](equation_solving.ipynb)
* [Ch.6 - Optimization](optimization.ipynb)
